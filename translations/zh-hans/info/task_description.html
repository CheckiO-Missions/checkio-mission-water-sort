Water Sort game 中会给你一连串烧瓶。其中大部分装满了不同颜色的混合液体，还有一些是空的。您的任务是使用空烧瓶收集烧瓶中相同颜色的液体，方法是从一个烧瓶中倒入单一颜色的液体到另一个烧瓶中，或倒空一个烧瓶。
