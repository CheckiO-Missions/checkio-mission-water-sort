In Water Sort game erhalten Sie eine Reihe von Fläschchen. Die meisten von ihnen sind mit gemischten Flüssigkeiten verschiedener Farben gefüllt, einige sind leer. Ihre Aufgabe ist es, Flüssigkeiten der gleichen Farbe in einem Kolben zu sammeln, indem Sie leere Kolben verwenden, indem Sie eine einzelne Farbportion(en) von einem Kolben in einen anderen mit dieser Farbe oben drauf gießen oder einen leeren Kolben.

Die Reihenfolge der Fläschchen wird als {% if is_js %}array{% else %}list{% endif %} von Strings dargestellt. Alle nicht leeren Fläschchen in einer Folge haben das gleiche Fassungsvermögen (das von Test zu Test variieren kann). Die Farben der Flüssigkeiten werden mit lateinischen Buchstaben kodiert, und zwar von oben nach unten (in den Kolben) und von links nach rechts in der Reihenfolge des ersten Auftretens (die erste Farbe wird als "a" kodiert, die nächste Farbe - "b" usw.). Benachbarte Teile der gleichen Farbe werden mit einem sich wiederholenden Buchstaben kodiert ("aa", "ccc" usw.).

Ihre Aufgabe ist es, die minimale Anzahl der Schüttungen zu berechnen, die erforderlich ist, um jede einzelne Farbe in einem Kolben zu sammeln.
